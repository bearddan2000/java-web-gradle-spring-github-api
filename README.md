# java-web-gradle-spring-github-api

## Description
A portal to classify all my repos.

## Tech stack
- java
- concurrent
  - threadpool
- generics
- gradle
  - spring
  - logback
  - failsafe
  - surefire
  - junit5
  - serenity
    - cucumber
    - springboot test

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
